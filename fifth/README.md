## [subjectPowerUp](subjectPowerUp.user.js?raw=true)

条目页面左下角观看信息优化，允许用户切换所有用户的观看信息/仅好友的观看信息

有对动画播放状态做判断，放送中的动画显示的是在看信息，而放送结束的动画则显示看过信息

暂时没有对书籍等进行判断，除了动画之外别的全是默认放送完毕只显示看过/读过/听过/玩过信息（显示为“看过”，这一点日后修复
